## 🔄 **Migration Guide**

### **For Existing Users:**

1. **📥 Download the New Application:**
   ```bash
   git clone https://github.com/pparesh25/NSE_BSE_Downloader.git
   cd NSE_BSE_Downloader
   ```

2. **⚙️ Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **🚀 Run the Application:**
   ```bash
   python main.py
   ```

### **Key Improvements Over Legacy Version:**
- ✅ **GUI Interface** instead of command-line only
- ✅ **Multiple exchanges** support in single application

- ✅ **User preferences** and settings persistence
